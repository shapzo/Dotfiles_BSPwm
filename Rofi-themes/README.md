In the following files: **`power.rasi`**, **`emoji.rasi`** and **`run.rasi`**, uncomment the lines according to the theme that was previously selected.

***Example***

1. For poly1 and poly3 theme
```bash
    x-offset:                       41.5%;
    y-offset:                       -5%;
```

2. For poly2 theme
```bash
    
    x-offset:                       37.4%;
    y-offset:                       -5%;
```
