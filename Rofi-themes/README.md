- The theme power menu

In the [power.rasi] file, uncomment the lines according to the theme that was previously selected.

1. For mac version
```http
    x-offset:                       41.5%;
    y-offset:                       -5%;
```

2. For windows version
```http
    
    x-offset:                       37.4%;
    y-offset:                       -5%;
    
    border:                         3px;
    border-color:                   @border-color;
```
